rfriends_lxcはxc環境でrfriends3を動作させるスクリプトです  
  
cd ~/  
wget -O rfriends_lxc-main.zip https://github.com/rfriends/rfriends_lxc/archive/master.zip  
unzip rfriends_lxc-main.zip  
cd rfriends_lxc-main  
sh rfriends3_lxc.sh
